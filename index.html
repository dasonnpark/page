Service Plan

JUST NEED A LOGO

플랜A. 로고 디자인 단독

₩300,000~

프로젝트 문의하기

포함 항목

  • 메인 로고 1안 + 서브 로고 (가로/아이콘형 1종)
  • PNG + JPEG + AI원본
  • 요청 시 흑백 버전 무료 제공

기타 사항

  • 작업 기간 : 5~7일
  • 수정 횟수 : 초안 1회 + 수정 2회 포함

권장 상황

  • 브랜드 명은 있지만 로고 디자인이 없는 경우
  • 기존 로고 디자인이 오래 되어 변경을 원하는 경우

FOR NEW BRANDS

플랜B. 네이밍 + 로고 디자인

₩500,000~

프로젝트 문의하기

포함 항목

  • 브랜드 네이밍 제안 5안 + 메인 로고 1안 + 서브 로고 1종
  • 네이밍 제안서 + 로고 파일(PNG/JPEG/AI원본) + 브랜드 스토리 PDF
  • 요청 시 브랜드 핵심 슬로건·소개글·스토리 제공 / ₩200,000 추가

기타 사항

  • 작업 기간 : 7~10일
  • 수정 횟수 : 네이밍 수정 1회 / 로고 수정 2회 포함

권장 상황

  • 브랜드 명부터 로고 디자인까지 필요한 경우
  • 기획력 있는 브랜드 명과 컨셉이 필요한 경우

*네이밍 구성 : 네임 아이디어 5개 + 각 이름의 컨셉/의미 설명

FOR ALL-IN-ONE BRANDING

플랜C. 브랜딩 런칭 패키지

₩1,000,000~

프로젝트 문의하기

포함 항목

  • 메인 로고 1안 + 서브 로고 1종 + 응용 시안 + 제작 요청용 매뉴얼 + 브랜드 가이드
  • 네이밍 제안서 + 로고 파일(PNG/JPEG/AI원본) + 브랜드 스토리 PDF

기타 사항

  • 작업 기간 : 2주 내외
  • 수정 횟수 : 네이밍 수정 1회 / 로고 수정 2회 포함

권장 상황

  • A-Z까지 로고 및 각종 인쇄물 등 통일성 있는 디자인이 필요한 경우
  • 간판, 인쇄물 등 바로 주문할 수 있는 가이드가 필요한 경우

*응용 시안 : 로고가 실제로 쓰이는 이미지 3종(간판, 스티커, 인스타 등)
*제작 요청용 매뉴얼 : 추후 디자이너나 간판집에 외주 요청 시 그대로 줄 수 있는 매뉴얼 제공
*브랜드 가이드 : 브랜드 컬러 코드 / 폰트 / 로고 여백 기준 / 로고 활용 가이드 등
